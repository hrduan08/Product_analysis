常见问题

试用期还在进行，可以直接升级吗？

可以，付款后立即转入正式套餐，并保留剩余试用时长。

完成支付后在哪里查看状态？

系统会跳回控制台“我的订阅”卡片，展示新套餐与倒计时。

如何取消自动续费？

在控制台的订阅页即可关闭自动续费，当前周期到期后停止扣款。
