Privacy Policy / 隐私政策

VoiceInsight Chrome Extension (YouTube subtitle downloader)

Effective Date / 生效日期：2026-02-17

English 中文

English

1. What this extension does

VoiceInsight helps users download subtitles/transcript text from the currently open YouTube video page and save them as a local .txt file.

2. Data we process

  • Website content: subtitle/transcript text available on the currently open YouTube video page
  • Browsing/network data: current YouTube video URL, page title, and subtitle request URLs needed to retrieve subtitles
  • User activity data: user-triggered download task status (running/success/failure)
  • Local extension state in chrome.storage.local (for example language preference and download task status/progress)
  • Local subtitle file content saved to the user’s Downloads folder

3. How we use data

  • Single-purpose use only: data is used solely to download subtitles for the current YouTube video that the user selects
  • Identify the current YouTube video and locate available subtitle/transcript text
  • Generate and save subtitle .txt files locally in the Downloads folder
  • Show task progress and status in the extension UI
  • No data is used for ads, profiling, or model training

4. Data sharing and selling

  • We do not sell user data.
  • We do not share user data for advertising or profiling.
  • We do not use user data for unrelated purposes.
  • Subtitle requests are made from the user’s browser to YouTube resources.
  • We do not send subtitle content to VoiceInsight servers in this workflow.

5. Permissions and purpose

  • activeTab: access the current tab after user action
  • scripting: run packaged scripts on YouTube pages to read subtitles
  • storage: save local settings and task status
  • downloads: save subtitle files to the Downloads folder
  • Host permissions (https://*.youtube.com/*, https://youtu.be/*) for subtitle retrieval on YouTube pages

6. Remote code

VoiceInsight does not use remote executable code. Extension JavaScript is packaged in the extension bundle submitted for review.

7. Data retention and user control

  • Local extension state remains until users clear extension data or uninstall.
  • Downloaded subtitle files remain on device until users delete them.
  • Users can stop using and uninstall the extension at any time.

8. Children

This extension is not intended for children under 13.

9. Changes to this policy

We may update this policy from time to time. Updates will be posted on this page with a revised effective date.

10. Contact

For privacy questions, contact: contact@voiceinsight.cloud

中文

1. 扩展功能

VoiceInsight 用于在用户当前打开的 YouTube 视频页面中下载可用字幕/转录文本，并保存为本地 .txt 文件。

2. 我们处理的数据

  • 网站内容：当前打开的 YouTube 视频页面中可用的字幕/转录文本
  • 浏览/网络数据：当前 YouTube 视频 URL、页面标题、以及下载字幕所需的请求 URL
  • 用户活动数据：用户触发下载后的任务状态（进行中/成功/失败）
  • 本地扩展状态（存储在 chrome.storage.local，例如界面语言偏好、下载任务状态/进度）
  • 保存在用户“下载”目录中的本地字幕文件内容

3. 数据用途

  • 单一用途：仅用于完成用户当前选择的 YouTube 视频字幕下载功能，不用于无关目的
  • 识别当前 YouTube 视频并定位可用字幕/转录文本
  • 在本地“下载”目录生成并保存字幕 .txt 文件
  • 在扩展界面中显示任务进度与状态
  • 不会将数据用于广告投放、用户画像或模型训练

4. 数据共享与出售

  • 我们不会出售用户数据。
  • 我们不会将用户数据用于广告投放或用户画像，也不会向第三方提供此类用途的数据。
  • 我们不会将用户数据用于与字幕下载无关的用途。
  • 字幕请求由用户浏览器直接请求 YouTube 相关资源以完成功能。
  • 在本扩展流程中，我们不会将字幕内容上传到 VoiceInsight 服务器处理。

5. 权限及用途

  • activeTab：在用户触发操作后访问当前活动标签页
  • scripting：在 YouTube 页面执行随扩展打包的脚本以读取字幕相关内容
  • storage：保存本地设置与任务状态
  • downloads：将字幕文件保存到“下载”目录
  • 主机权限（https://*.youtube.com/*、https://youtu.be/*）用于访问下载字幕所需的 YouTube 页面/资源

6. 远程代码

VoiceInsight 不使用远程可执行代码。扩展 JavaScript 均打包在提交审核的扩展安装包内。

7. 数据保留与用户控制

  • 本地扩展状态会保留在浏览器中，直到用户清除扩展数据或卸载扩展。
  • 下载后的字幕文件会保留在用户设备中，直到用户自行删除。
  • 用户可随时停止使用并卸载扩展。

8. 儿童隐私

本扩展不面向 13 岁以下儿童。

9. 政策更新

我们可能不定期更新本政策。更新后将发布在本页面，并更新生效日期。

10. 联系方式

如有隐私相关问题，请联系： contact@voiceinsight.cloud
